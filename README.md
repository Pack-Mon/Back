| 타입 | 설명 |
		| --- | --- |
| Feat | 새로운 기능 추가 |
| Fix | 버그 수정 |
| Refactor | 리펙토링 |
| Design | CSS, UI 디자인 변경 |
| Style | 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우 |
| Test | 테스트 코드 추가, 수정, 삭제 (비즈니스 로직 변경 없음) |
| Chore | 기타 변경사항 (빌드, 패키지 매니저 등) |
| Init | 프로젝트 초기 생성 |
| Rename | 파일 또는 폴더명 수정 |
| Remove | 파일 삭제 작업만 수행 |
